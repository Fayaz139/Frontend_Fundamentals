# Contact Sales Page UI (HTML & CSS)

This project is a clean and modern **Contact Sales page UI** built using **HTML and CSS** only.

## Features
- Two-column layout (content + form)
- Flexbox-based responsive structure
- Vertically centered form section
- Step progress indicator beside action button
- Sticky footer on the left panel
- CSS-only animated gradient footer

## Technologies Used
- HTML5
- CSS3 (Flexbox, Animations)

## Purpose
This project was created as part of **Frontend Fundamentals practice** to improve layout design, spacing, alignment, and real-world UI cloning skills.
