# Login Page UI (HTML & CSS)

This project is a simple and responsive **Login / Sign Up page UI** built using **HTML and CSS** only.

## Features
- Centered login layout using Flexbox
- Google Sign Up button
- Email and Password input fields
- Clean footer with Terms & Privacy links
- Fully responsive (desktop & mobile)

## Technologies Used
- HTML5
- CSS3 (Flexbox)

## Purpose
This project was created as part of **Frontend Fundamentals practice** to understand layout structuring, spacing, and basic form UI design.
